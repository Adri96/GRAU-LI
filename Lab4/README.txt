Cada problema tiene su solución en su propio directorio. 
Para crear el ejecutable es necesario correr el comando make desde el directorio correspondiente.
El ejecutable tendrá el mismo nombre que el fichero .pl.

Para el problema de las charlas, intenté 2 soluciones diferentes:
	- Backtracking sobre las asignaciones de slots para cada estudiante (charla.pl).
	- Backtracking sobre los slots de cada grupo (charlaBig.pl).

La primera es la más rápida, y es la que se genera con el make.
